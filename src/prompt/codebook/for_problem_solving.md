# For Problem-solving

## Definition
A goal of using a model to guide problem-solving and decision making. The model
helps stakeholders identify what needs to change or make a decision, but the
model itself is NOT the implementation instrument.

## Signal patterns
- Positive: decision, analyze options, evaluate, identify shortcomings,
  what should change
- Negative: directly implemented after, used to build

## Disambiguation
**vs. For Intervening:** If the model is only used to identify
shortcomings/problems/improvements or to make a decision -> Problem-solving. If
the change is reflected in a built schema/system/artifact -> Intervening. Code
both only when separate evidence shows both decision-making and implementation.

## Confidence
Present only at High or Medium.
